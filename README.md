# muzz-livevideo

Document de pitch HTML statique pour le concept d'app de double-dating live vidéo
(nom de travail : **Mute**).

## Structure

- `index.html` — document complet, autonome (Tailwind via CDN, Google Fonts)
- Pas de build, pas de dépendances. Ouvre `index.html` directement ou déploie tel quel.

## Déploiement

Hébergé sur Vercel comme site statique. Tout le HTML est dans `index.html`
à la racine — Vercel le détecte et le sert automatiquement.
